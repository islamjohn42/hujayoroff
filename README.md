## 🚀 How to Use Language Switching for SEO

### URL-Based Language Switching
```
https://www.hujayoroff.uz/?lang=en  (English)
https://www.hujayoroff.uz/?lang=ru  (Russian)
https://www.hujayoroff.uz/?lang=uz  (Uzbek)
```

### Features
- Language preference is saved in localStorage
- URL automatically updates when user switches language
- Search engines recognize each language version separately
- hreflang tags guide search engines to appropriate versions
